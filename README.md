## ¿Cómo ejecutar la aplicación?
Lo primero, será asegurarno que tengamos instalado el **.Net Core 3** instalado, para ello realizaremos lo siguiente:

1. Abrir la terminar de windows y ejeuctar el comando **dotnet --version**
2. Si el resultado de eso es: "dotnet no se reconoce como un comando interno o externo, proframa o archivo por lotes ejecutables", entonces es necesario instalar el **SDK de .Net Core**, [x64](https://dotnet.microsoft.com/download/thank-you/dotnet-sdk-3.0.100-windows-x64-installer) - [x82](https://dotnet.microsoft.com/download/thank-you/dotnet-sdk-3.0.100-windows-x86-installer).
3. En caso de que este instalado el mensaje deberia de ser: **3.0.100**.
4. Tener [**Visual Studio Code**](https://code.visualstudio.com/)
